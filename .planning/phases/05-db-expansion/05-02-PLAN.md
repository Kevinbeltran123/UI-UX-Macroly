
**Query 2 — Verificar que ningún producto tiene dietary_tags vacío:**
```sql

FROM public.products
WHERE dietary_tags = '{}' OR dietary_tags IS NULL;
```
Resultado esperado: 0

**Query 3 — Verificar que todos los meal_context son valores válidos:**
```sql
SELECT COUNT(*) AS invalid_meal_context
FROM public.products
WHERE meal_context NOT IN ('any', 'breakfast', 'lunch', 'dinner');
```
Resultado esperado: 0

**Query 4 — Smoke test vegano (≥5 resultados):**
```sql
SELECT COUNT(*) AS vegano_count
FROM public.products
WHERE dietary_tags @> ARRAY['vegano'];
```
Resultado esperado: ≥ 5

**Query 5 — Smoke test breakfast (≥5 resultados):**
```sql
SELECT COUNT(*) AS breakfast_count
FROM public.products
WHERE meal_context = 'breakfast';
```
Resultado esperado: ≥ 5

**Query 6 — Smoke test sin_gluten (≥5 resultados):**
```sql
SELECT COUNT(*) AS sin_gluten_count
FROM public.products
WHERE dietary_tags @> ARRAY['sin_gluten'];
```
Resultado esperado: ≥ 5

**Query 7 — Verificar que los 12 productos existentes tienen tags actualizados:**
```sql
SELECT name, dietary_tags, meal_context
FROM public.products
WHERE name IN (
  'Pechuga de Pollo','Arroz Integral','Huevos AAA x12','Avena en Hojuelas',
  'Atun en Agua','Banano x6','Aguacate Hass x2','Leche Deslactosada',
  'Pan Integral','Proteina Whey','Yogur Griego','Pasta Integral'
)
ORDER BY name;
```
Resultado esperado: 12 filas, ninguna con `dietary_tags = '{}'`

Ejecutar con:
```bash
supabase db query --query "SELECT COUNT(*) AS total_products FROM public.products;"
```
o alternativamente copiar las queries en Supabase Studio > SQL Editor si `supabase db query` no está disponible.

Registrar todos los resultados en el SUMMARY.
